## Estimate divergence-diversity ratio (*D*) for homologous TR loci

### scripts:
- `get_tr_ratios.R`: Calculate HKA-like divergence-diversity TR ratio (*D*) from allele length data. Generated output from human-chimp data is stored at `variable_trs_ratio.txt`
- `plot_ratios.R`: Takes `variable_trs_ratio.txt` as input and generate ratio figures.

### files:
- `variable_trs_ratio.txt`: Contains genomic location, characterization and divergence-diversity (*D*) values for human-chimp TR variants
